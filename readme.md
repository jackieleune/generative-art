## Concept

Ik wil met mijn concept mezelf uitstralen. Hoe ik mezelf graag zie en hoe ik in het leven probeer te staan. Dat is vrolijk, positief en kleurrijk.
Net zoals hoe ik bloemen zie, deze werden dan ook al snel het onderdeel waar mijn concept omheen hing. Toch weet iedereen dat dit niet altijd zo realistische is. Dit kan je merken als je interactie hebt met de website. Zodra je te dicht bij de zon komt (hier overheen hovert) wordt hij donker. 

## Features

- Een wolk die bovenaan het scherm heen zweeft 
- De zon waarvan de stroke veranderd als je op de muis klikt
- De zon die van kleur/fill veranderd als je er overheen hovert
- De 2 buitenste bloemen die een random grootte krijgen elke keer als de pagina opnieuw laadt
- De bewegende glitterbal die aan de muis vastplakt
